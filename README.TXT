PROJECT TITLE: Capacity Coach Shipping

PURPOSE OF PROJECT: Each town has a certain capacity limit set. Once one is reached for one destination, that town is full and dispatched.
The user can fill the coaches with letters or packages until full.

VERSION or DATE: 9/30/2022 11:34 am

HOW TO START THIS PROJECT: Simply hit run to start 

AUTHORS: Collins Ramsey, Dr. Ellie Lovellette

USER INSTRUCTIONS: Follow the instructions of the program as it goes along once started. 
You must enter a valid town name, with correct spelling. 
Then enter a type of mail, which can be package, packages, letter, or letters.
The last keyboard input is the amount you want to send. Keep in mind the number you type for 
packages is mutltiplied by 20.
Fill each town coach and see what happens!
